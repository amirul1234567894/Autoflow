What Is n8n?

n8n (pronounced "n-eight-n") is a workflow automation tool that lets you connect different apps and automate tasks between them. Think of it like a smart assistant that watches for events — like a new Telegram message or a new row in Google Sheets — and automatically takes action.

The big difference between n8n and tools like Zapier or Make is that n8n is open-source and self-hosted. This means:

  • No expensive dollar-based subscriptions — you pay for your own server (₹200–500/month), not per workflow run
  • Your data stays on your server — complete privacy and control
  • Unlimited workflows — no artificial caps on how many automations you can run

💡 Why n8n is perfect for India: Most automation tools charge in USD ($20–100/month). n8n running on a cheap Indian VPS costs ₹200–500/month — and can run unlimited workflows. The ROI is unmatched.

Setting Up Your First n8n Workflow

Here's how a simple lead capture workflow looks in n8n:

Trigger: Telegram Bot — new message received
↓
Node 1: Extract name, phone, service from message
↓
Node 2: Google Sheets — append new row with lead data
↓
Node 3: Telegram — send confirmation reply to customer
↓
Node 4: Telegram — notify you with lead details
↓
✅ Done — entire flow runs in under 1 second

Open the n8n editor, drag in a Telegram Trigger node, connect it to a Google Sheets node and a Telegram Send Message node. Set up the data mapping and activate the workflow.

Common Questions

Do I need to know how to code?

No. n8n has a visual drag-and-drop interface. You connect nodes (like building blocks) and configure them through forms — no programming knowledge needed.

What's the monthly cost?

A basic VPS to host n8n costs ₹200–500/month. n8n itself is free and open-source. That's it. Compare that to Zapier at $20–50/month with strict limits.

Is it reliable?

Yes — n8n runs 24/7 on your server and executes workflows in under a second. If a workflow fails, you get an error notification so you can fix it immediately.
