How to Build This in n8n

  1. Trigger: Telegram message received OR form webhook
  2. Save to Sheets: Name, phone, requirement, timestamp, status = "New"
  3. Send instant reply via Telegram or SMS
  4. Wait 24 hours — n8n's Wait node handles this natively
  5. Check status in Sheet — if still "New" (not converted), send Follow-up #1
  6. Wait 48 more hours → if still "New", send Follow-up #2

💡 Key insight: The status column in your Google Sheet is what makes this smart. When you close a deal, you change the status to "Converted" — the workflow sees this and skips the follow-ups for that lead. Clean and automatic.

Common Mistakes to Avoid

  • Don't over-automate: Use automation for the first 2–3 touchpoints. After that, a personal call works better.
  • Personalise the messages: Use the lead's name and reference what they asked about. Generic messages get ignored.
  • Set a clear CTA: Every message should tell the lead exactly what to do next — book a call, reply with yes, visit a link.
  • Track your numbers: Use the Google Sheet to monitor how many leads convert after follow-up #1 vs #2. Optimise over time.
